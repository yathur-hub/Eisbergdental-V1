# Eisberg Dental — Frontend Solution

Dieses Repository enthält die Benutzeroberfläche für das zahntechnische Labor **Eisberg Dental** in Zürich. Die Anwendung ist in React, Vite und Tailwind CSS implementiert.

---

## 🔒 Wichtiger regulatorischer Hinweis

Sämtliche Benutzeroberflächen wurden gemäss der Schweizer Gesetzgebung, insbesondere der **Medizinprodukteverordnung (MepV)** und dem **Datenschutzgesetz (DSG)**, bereinigt. Es wurden alle unbelegten oder ungeprüften Behauptungen, Zertifizierungen, Garantieversprechen und Testimonials entfernt oder neutralisiert.

* **Interne Abklärungen & TODOs**: Alle Punkte, die eine geschäftliche Entscheidung oder eine juristische Prüfung durch den Eigentümer Thomas Barandun erfordern, sind in der Datei [INTERNES-TODO.md](./INTERNES-TODO.md) festgehalten.
